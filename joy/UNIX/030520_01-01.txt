유닉스의 특징과 역사
- 컴퓨터 시스템을 효율적으로 사용하기 위한 운영체제
- 특징
  - 대화형 시스템
  - 다중 사용자 시스템
  - 다중 작업용 시스템
  - 높은 이식성과 확장성
  - 계층적 파일 시스템
  - 다양한 부가적 기능 제공

구조
 - 커널
  - 핵심 기능
   - 프로세스 관리
   - 메모리 관리
   - 파일 시스템 관리
   - 장치 관리
   : 컴퓨터의 모든 자원 관리
 - 셸
  - 사용자와 커널 사이의 중간자 역할
  : 사용자(명령) -> 셸(해석) -> 커널(수행) -> 셸(결과 출력)
 - 유틸리티

 * 기초 명령의 기능
  - banner : 문자열을 큰 문자로 출력
  - date : 현재 날짜와 시간 출력
  - clear : 화면을 깨끗이 지워줌
  - man : 명령에 대한 자세한 설명 출력
  - passwd : 비밀번호 변경
